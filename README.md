#### Описание:
Проект представляет собой реализацию алгоритма кластерного анализа для проведения кластеризации множества переменных по нескольким признакам. Кластером является совокупность пеерменных со схожими значениями признаков. В рамках проекта были прописаны функции, являющиеся этапами кластерного анализа:
1. Создание тестового массива данных (generate_data);
2. Загрузка данных для анализа (load_data);
3. Подготовка данных для анализа (prepare_data);
4. Нахождение оптимального количества кластеров методом локтя (find_best_k);
5. Кластеризация данных с помощью метода k-средних (cluster_data);
6. Визуальное представление разбиения на кластеры (visualize_clusters);
7. Расчет статистик по кластерам (show_statistics);
8. Сохранение результата кластерного анализа (save_results).

Со стороны пользователя проект представляет сценарий, который последовательно выводит этапы кластерного анализа предоставляя возможность пользователю оределять источник данных (тестовые данные или собственные исходные данные), количество кластеров и вывод результатов анализа в файл. В рамках анализа пользователю демонстрируются статистические метрики по каждому из признаков, графическое представление метода локтя для принятия решения о количестве кластеров исходя из графического элемента, графическое представление кластерного анализа на графике, значения статистических метрик по кластерам.

Проект применим в рамках анализа большого количества переменных со множеством различных показателей с целью их разбиения на схожие группы. Широкое распространение данный проект может получить в рамках научных исследований. Например, оценка ценных бумаг, социально-экономическая дифференциация регионов и др.
